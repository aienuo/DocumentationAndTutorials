# 域名证书自动更新 #




[阿里云域名配置参考](https://blog.csdn.net/weixin_43844718/article/details/147043427)

[其他参考](https://atpx.com/blog/auto-update-ssl-with-acme/)

## 安装 `acme.sh` ##

```shell
curl  https://get.acme.sh | sh -s email=ienuo@qq.com
```

## 刷新 `shell` 变量配置 ##

```shell
source ~/.bashrc
```

## 导入 阿里云的 `Key` 和 `Secret` ##

```shell
export Ali_Key=""
export Ali_Secret=""
```

## 设置默认 `CA` 为 `Let's Encrypt` ##

```shell
acme.sh --set-default-ca --server letsencrypt
```

## 签发 `RSA` 证书 ##

```shell
acme.sh --issue --dns dns_ali -d aienuo.com -d *.aienuo.com --keylength 2048
```

## 安装 `RSA` 证书到指定路径 ##

```shell
acme.sh --install-cert -d aienuo.com \
--key-file        /usr/local/nginx/cert/aienuo.com_private.key  \
--fullchain-file /usr/local/nginx/cert/aienuo.com_chain.pem \
--capath         /usr/local/nginx/cert/aienuo.com_ca.pem  \
--reloadcmd      "sudo systemctl restart nginx"
```

## 签发 `ECC` 证书，若不需要，跳过即可 ##

```shell
acme.sh --issue --dns dns_ali -d aienuo.com -d *.aienuo.com --keylength ec-256
```

## 安装 `ECC` 证书到指定路径 ##

```shell
acme.sh --install-cert -d aienuo.com --ecc \
--key-file       /usr/local/nginx/cert/ecc_aienuo.com_private.key  \
--fullchain-file /usr/local/nginx/cert/ecc_aienuo.com_chain.pem \
--capath         /usr/local/nginx/cert/ecc_aienuo.com_ca.pem  \
--reloadcmd      "sudo systemctl restart nginx"
```

## 取消环境变量 ##

```shell
unset Ali_Key
unset Ali_Secret
```

## 查看证书列表 ##

```shell
acme.sh --list
```

## 查看证书信息 ##

```shell
acme.sh --info -d aienuo.com
```

## 手动续期证书 ##

```shell
acme.sh --renew -d aienuo.com
```

## 手动删除证书 ##

```shell
acme.sh --remove -d aienuo.com
```
